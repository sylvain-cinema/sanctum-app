## Overview

SANCTUM is the member app for Sylvain's highest tier — ultra-exclusive private cinema installations with bespoke 16K MicroLED, Sonora Elite audio, and full Stratum volumetric integration. Features Web3 token-gated access via Aethelred, concierge booking, private screening management, and curated member experiences.

## Features

- **Token-Gated Access**: Connect Web3 wallet to verify SANCTUM NFT membership
- **Private Screenings**: Book personal 16K cinema sessions
- **Concierge Service**: AI-powered personal cinema concierge
- **Member Directory**: Connect with fellow SANCTUM members
- **Exclusive Content**: Early access to Sylvain-mastered premieres

## Tech Stack

- React Native / Expo Router
- TypeScript
- Web3 wallet integration (Aethelred)
- Sylvain SDK for venue control
