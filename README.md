## Features

- **Token-Gated Access** — Connect Web3 wallet to verify SANCTUM NFT membership
- **Private Screenings** — Book personal 16K cinema sessions
- **Concierge Service** — AI-powered personal cinema concierge
- **Member Directory** — Connect with fellow SANCTUM members
- **Exclusive Content** — Early access to Sylvain-mastered premieres
- **Glassmorphism UI** — Sylvain dark + gold design language

<br/>
